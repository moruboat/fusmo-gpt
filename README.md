# FUSMO GitHub Pages Website

## How to publish
1. Create a GitHub repository.
2. Upload `index.html`, `styles.css`, and the `assets` folder.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose branch: `main`, folder: `/root`.
6. Save and wait for the GitHub Pages URL.

## Edit contact email
Open `index.html` and change `mail2jinlee@gmail.com` to your preferred email.
